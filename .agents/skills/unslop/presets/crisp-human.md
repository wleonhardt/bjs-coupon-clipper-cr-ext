# Preset: Crisp Human

Short, direct, no fluff. Every word earns its place.

---

## Voice Characteristics

- **Sentence length**: 5-15 words preferred. Max 20.
- **Paragraph length**: 1-3 sentences. White space is your friend.
- **Tone**: Confident. No hedging.
- **Structure**: Lead with the point. Support briefly. Stop.

---

## Rules

1. **Cut ruthlessly** - If removal doesn't change meaning, remove it
2. **One idea per sentence** - No compound thoughts
3. **Active voice only** - Subject → verb → object
4. **No filler adverbs** - "Very," "really," "truly," "just" → delete
5. **Concrete over abstract** - "Sent three emails" not "communicated extensively"

---

## Structural Patterns

### Opening
```
[Main claim/fact/insight]
```
Not:
```
[Context] + [Setup] + [Main claim]
```

### Supporting
```
[Evidence or example]. [Implication].
```
Not:
```
[Transition] + [Evidence] + [Explanation of evidence] + [Why it matters]
```

### Closing
```
[Implication or call to action]
```
Not:
```
[Summary] + [Restatement] + [Final thought] + [Kicker]
```

## Examples

### Before (AI)
> In today's fast-paced business environment, it's becoming increasingly important for organizations to focus on what truly matters: their customers. When companies take the time to deeply understand customer needs, they position themselves for long-term success.

### After (Crisp)
> Companies that know their customers win. The rest guess.

---

### Before (AI)
> I think it's fair to say that the most successful teams are those that have learned to embrace failure as a learning opportunity rather than viewing it as something to be avoided at all costs.

### After (Crisp)
> Good teams fail fast. Bad teams avoid failure.

---

### Before (AI)
> If you're looking to improve your writing, there are several key strategies you should consider implementing. First, focus on clarity. Second, eliminate unnecessary words. Third, use active voice whenever possible.

### After (Crisp)
> Better writing: be clear, cut words, use active voice.

---

## Quality Checklist

- [ ] First sentence contains the main point
- [ ] No sentence exceeds 20 words
- [ ] No paragraph exceeds 3 sentences
- [ ] Zero hedging words (think, maybe, perhaps, might)
- [ ] Zero filler adverbs (very, really, truly, just)
- [ ] All verbs concrete and active
- [ ] Would sound natural read aloud

---

## Common Transformations

| From | To |
|------|-----|
| "It's important to note that" | [delete] |
| "In order to" | "To" |
| "The fact that" | [delete, restructure] |
| "Due to the fact that" | "Because" |
| "Is able to" | "Can" |
| "Has the ability to" | "Can" |
| "At this point in time" | "Now" |
| "In the event that" | "If" |
| "Prior to" | "Before" |
| "Subsequent to" | "After" |
| "A large number of" | "Many" |
| "The vast majority of" | "Most" |
